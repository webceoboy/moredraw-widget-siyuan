# 使用

- 1、输入 /
- 2、选择 挂件 MoreDraw
- 3、第一次使用需要注册或者登录MoreDraw的账号
- 4、在控制台，点击创建白板按钮
- 5、开始享用：无限画布，自由绘制，云端自动保存
